## 🕶️ 팀 소개 

### 안녕하세요, 팀 "따라따라" 입니다!

<br />

저희는 6주간 ⚒ **"기술적 도전"** ⚒을 목표로 삼았습니다.

<br />

저희는 단순히 api나 라이브러리를 **"가져다 써서" 구현 하는 것이 아닌**,

**모든 것을 "직접 구현"** 해보고, 그 과정에서 깊이 있는 기술적인 도전을 해 나갑니다.
